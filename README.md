# Sistema de Gestão de Multas para Frotas

## 📌 Problema
Empresas de transporte possuem dificuldade no controle de multas,
prazos e identificação de responsáveis.

## 🎯 Objetivo
Criar um sistema automatizado para:
- Registrar multas
- Gerar notificações
- Controlar status
- Emitir relatórios

## 🛠 Tecnologias
- Python
- PySide6
- SQLite
- Pandas

## ⚙ Funcionalidades
- Cadastro de multas
- Controle por motorista
- Geração de CSV
- Automação de status
- Sistema de filtros

## 🚀 Resultado
- Centralização das multas
- Redução de erros manuais
- Controle de prazos
